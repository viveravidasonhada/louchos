
# LaunchOS - Sistema Inteligente de Lançamentos

O LaunchOS é uma plataforma para Estrategistas de Lançamento que utiliza IA para gerar planos operacionais completos, desde a copy até a gestão de tarefas da equipe.

## Funcionalidades Principais

*   **IA Generativa (Gemini 2.5):** Cria estratégias baseadas em Blueprints comprovados (Semente, Interno, Meteórico, Perpétuo).
*   **Gestão de Experts:** Armazena o "cérebro" do expert (tom de voz, nicho) para personalizar a IA.
*   **Dashboard Operacional:** Visão Kanban, Timeline e Lista por Membro da equipe.
*   **Blueprints Editáveis:** Crie seus próprios modelos de lançamento e deixe a IA preencher as tarefas.
*   **Exemplos Práticos:** Cada tarefa gerada vem com 3 exemplos de execução (headlines, ideias de vídeo, etc.).

## Como Usar

1.  **Login:** Use `admin@launchos.ai` / `admin` para acesso total.
2.  **Cadastre a Equipe:** Vá em "Equipe" e adicione seus colaboradores.
3.  **Cadastre um Expert:** Vá em "Experts" e preencha o nicho e cole exemplos de textos para treinar o estilo.
4.  **Novo Projeto:** Clique em "Novo Lançamento", selecione o Expert e o Tipo de Campanha.
5.  **Execução:** Acompanhe o progresso no Dashboard, marque tarefas como concluídas e anexe arquivos.

## Instalação Local

1.  Clone o repositório.
2.  Instale as dependências: `npm install`
3.  Inicie o servidor de desenvolvimento: `npm run dev`
