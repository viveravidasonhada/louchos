
# Roadmap do Produto

Planejamento de funcionalidades futuras para o LaunchOS.

## Q4 2024 - Otimização de IA & UX
- [x] Integração com Gemini 1.5/2.5 Flash.
- [x] Blueprints dinâmicos editáveis.
- [x] Seletor de projetos multitarefa.
- [ ] Refinamento do Prompt para gerar copys completas de e-mail (não apenas resumos).

## Q1 2025 - Integrações & Automação
- [ ] **Integração com Trello/ClickUp:** Exportar tarefas geradas para ferramentas externas via API.
- [ ] **Integração WhatsApp:** Disparar mensagens de aviso para a equipe quando uma tarefa atrasar.
- [ ] **Geração de PDF:** Exportar o plano estratégico em um PDF profissional "White Label" para enviar a clientes.

## Q2 2025 - Dados & Analytics
- [ ] **Dashboard Financeiro:** Input de custos de tráfego e cálculo de ROI em tempo real.
- [ ] **Debriefing com IA:** Upload de CSV de vendas pós-lançamento para a IA analisar o que errou e sugerir melhorias para o próximo.

## Backlog Técnico
- [ ] Migrar autenticação simulada para Supabase Auth real.
- [ ] Implementar Row Level Security (RLS) no banco de dados para multi-tenancy (múltiplas agências usando o mesmo software).
- [ ] Testes automatizados E2E com Cypress.
